Design

This is a twelve columns CSS grid system. It is not a be all, end all grid solution but it is to help you quickly generate a structured layout for desktop, tablet and smartphones. By using relative perecentage width for the columns, this grid accommodates a variety of devices and their pixel densities. Font sizes will have to be tinkered so that it looks and feels right and at the ideal 50 to 80 character count per line.

Purpose

To reinvent the wheel in a sense. There are plenty of CSS grid system frameworks out there that are fixed, fluid, responsive or a combination. I wrote my own to build my own to build my skills, to further my understanding of fluid and responsive design and media queries. Besides understanding technical challenges of layouts for multiple types of devices, there is also the content strategy challenge.

Future

I will be using this in framework in my projects and hopefully others will, too and provide feedback and contribute. With retina or pixel dense display devices becoming more popular, this grid will have to be continually Eventually, I will create a SASS version of this to use variables to make the grid width more configurable and DRY up the code. The documentation and usage will probably have to be improved upon.
